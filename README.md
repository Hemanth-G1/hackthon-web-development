# ♻️ Reusable Plastic Awareness Website

## Overview

This project is a sustainability-focused website developed during a hackathon to promote awareness about reusable plastics and eco-friendly practices. The website educates users about the environmental impact of single-use plastics and encourages the adoption of reusable alternatives.

## Features

* Responsive and user-friendly design
* Information about reusable plastic products
* Environmental awareness content
* Sustainable living recommendations
* Clean and modern interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Objective

The primary goal of this project is to spread awareness about plastic waste reduction and encourage sustainable consumption through reusable plastic solutions.

## Project Structure

```text

style.css
script.js
assets/
```

1. Clone the repository:

   ```bash
   git clone https://github.com/Hemanth-G1/hackthon-web-development.git
   ```
